对程序员而言，尤其是android开发，网上出国留学重要程度不言而喻，随时要欣赏学习下不一样的人体艺术，哦不，是api文档。可以说这么说，不会科学上网都不好意思说自己是程序员，那就让我们来打造个科学上网利器吧。com bady，let’s go.

下载镜像

openwrt官网 转到Downloads，选择Stable Release builds，再选择最新版本，此时为18.06.5。选择target，再选择树莓派对应的brcm2078，选择pi3的bcm2710，下载rpi-3-ext4-factory.img.gz

gunzip解压出img

刷入sd卡

安装balenaEtcher，刷入镜像

插卡，连上显示器，通电，绿灯亮，网口和电脑直连，浏览器输入192.168.1.1，进入openwrt luci界面，默认账号为root root

配置v2ray core

本质是一个代理工具，sock、http代理，实现了加密等

GitHub项目地址：https://github.com/v2ray/v2ray-core

官网：https://www.v2ray.com

下载v2ray core

选择github release，写此文时的最新版本是v4.21.3，下载v2ray-linux-arm64.zip

上传、解压

​ 解压：zip到/root/v2ray/，也可以直接直接解压了scp上来
